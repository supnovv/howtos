BP/DS
```
[KICK/BP]
不带人声读"不"，可用"八百标兵奔北坡"无人声版练习唇齿

[HIHAT/DS]
不带人声读"D"，发音要轻松、干脆、有力
连续不带人声"dididi"，舌尖顶的力量要集中、快速、有力

[SNARE]
自然抿嘴，唇间形成一个小缝隙，吸气产生摩擦音，然后瞬间张开嘴巴
吸气时轻松地吸气到嘴巴就行，不需要吸气入肺
可以通过一点点地调整唇间缝隙的位置和大小来尝试找到发出摩擦音的方法
```
