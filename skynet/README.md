
## Build
```shell
$ git clone git@github.com:cloudwu/skynet.git
$ cd skynet
$ git submodule update --init
$ make linux   # or macosx
# start the server
$ ./skynet examples/config
# open a new terminal and launch a client
$ ./3rd/lua/lua examples/client.lua   # and try input sth.
```
