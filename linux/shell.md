
**zip**
```shell
$ zip dest.zip file1 file2 file3  # compress specified files
$ zip dest.zip -r folder/         # compress all files in folder
$ unzip dest.zip                  # extract to current folder
$ unzip dest.zip -d folder/       # extract to specified folder
```
