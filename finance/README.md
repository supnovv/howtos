**目标市值策略**

恒生低估时做了目标市值，现在红利比较低估是否可以做目标市值呢？不是不可以做，
只是它的收益率不好确定。而且目标市值这个策略，我始终认为先要考虑A股的全市场
估值，至少在全市场不贵的地方开始。因为目标市值要投入一笔较多的资金，相对来说
比较危险。

目标市值策略的一个重要问题是如何确定目标涨幅。养老16%已经是一个非常高的目标收益率了，我建议这样的收益率在估值极低，
也就是15%到20%时可以考虑。另外在估值极低的时候进行目标市值，最关键的是操作时间不能是一个月，至少是三个月到半年。
我对目标市值的理解就是一次投入大量资金才会有好的效果，所有对估值的要求就更加严格。

**关于定投和择股**

从过往经验看，除非极度高估、即不特别贵、即没有便宜品种但有品种不是特别贵的时候，每月还是至少买一份，即大的原则是不择时、按时买。
但A股市场太过变态，有时候买入长期看收益真的很差，因此我们才适时稍微择一下，到了某个仓位阶段就大笔买入一次。

关于投资标的的选择，买书学习看财报是一条路，但其实也有两条更简单的道路。第一，定时定额定投指数基金，超越市场平均水平。
定时，放弃择时。定额，固定金额保证了越跌买越多，越涨买越少的基本投资原则。指数基金，代表平均水平，通过定时定额定投指数
基金，基本可以确定超越平均水准。第二，格雷厄姆给业余投资人的四项基本原则：1）适当但不要过分的分散，持股不少于10只，不多于30只；
2）只持有大型的、知名的、在财务上是稳健的公司；3）每家公司都应具有长期连续的支付股息的记录；4）买入股票的价格限制在每股平均
利润的25倍以内，每股平均利润，取过去7年的平均数。简单吧，这也许就是格老说的“少许的努力和能力”吧。

投资这件事有个特点，门外汉只需少许的努力和能力，便可以达到令人敬佩的结果，但如试图超越这唾手可得的成就，便需要付出无比的智慧
和精力。如果你希望稍微改善一下正常的绩效而在你的投资策略中加入一些额外的知识和技巧，你会发现－你反而退步了~~~　本杰明.格雷厄姆

**定向增发**

上市公司可以专门发行一批股票，卖给指定的对象（例如核心员工或核心经销商），
定增价格要比市场价格便宜很多。按照以往的规定，买入定增股票后要持有一年才能
卖掉，所以它又叫“限售股”。即使限售还是有很多人想要，毕竟便宜。一来二去，
就有了一种投资定增项目的基金，称为定增基金。

但端午节前，证监会出了新规，在解禁后也不能随便卖了，解禁后一年之内最多卖掉
持有的一半，而且每三个月卖掉的不能超过总股票的1%，等等。简而言之，再也不许
一夜清仓了，得细水长流慢慢卖，以便减少对市场的冲击。

这对定增基金来说，就很尴尬。原本只要忍12个月就好，但新规之后，基金买入定增
股，要在手里拿两年半以上才能全部卖掉。而现在市场上的定增基金到期后，很多基
民会选择赎回，到时候基金手上的定增股只让卖一半，好尴尬。

可转债（Convertible Bond）是可转换公司债券的简称，可以转换为债券发行公司的股
票。本质上讲，可转债是在发行公司债券的基础上，附加了一份期权，允许购买人在规定的
时间范围内、按特定条件将其购买的债券转换成指定公司的股票。

公司为什么要发行可转债呢？因为可转债相对普通债券和贷款，票面利率低，而且未来如果
股价表现好，则转股后连利息都不要付，降低了公司的融资成本。另外相对于定增非公开等
股权融资，转债对每股盈利的稀释效应比较缓和；再者，转债条款灵活，可根据自身情况和
市场需求量身定做。

**两融账户**

用两融账户配置基本不动的仓位，然后用费率非常低的普通账户做市值买卖。
这样基本仓位可以做担保品，以后大底可以从容融资，买卖的部分手续费又很低。

---

两融业务指的是融资融券交易。融资交易是投资者以资金或证券作为质押，向券商借入资金
用于证券买卖，并在约定期限内偿还借款本金和利息。融券交易是投资者以资金或证券作为
质押，向券商借入证券卖出，在约定的期限内，买入相同数量和品种的证券归还券商并支付
相应的融券费用。

## 认知与践行

做任何事情多要避免盲目，学习也一样。　　
动机大于一切，有了强烈的动机，无所谓专业知识，外行也可以很快的学会一个领域的知识。　　

**投资世界里最基本技能－趋势判断**

能证明自己对未来的判断是正确的非常难，因为：
1. 要等到很久很久以后，才有确定结果
2. 即便判断是对的，现实也不一定会当场给你正面反馈
3. 即使结果站在你这一边，你也不能保证没有运气因素的存在，完全都是你的判断在起作用
4. 只有正确判断本身是不够的，如果没有不顾一切的践行，最后你怎们证明你是对的呢

首先，让我们深入研究一个看似简单，貌似谁都早就明白的概念：周期。

周期，是理财投资活动中最为关键的考量因素，是开始实践之前必须学习、研究、掌握、遵循
的理念和现实。不深入理解周期，就无法对趋势进行有效的判断，那么整个投资活动基本上就
等于是没有判断的行为，于是甚至比不过两个人抛硬币赌博，而这时墨菲定律一定会显灵：

如果一件事有变坏的可能，那么它一定会变坏／如果有坏事发生的可能，那么坏事一定会发生。

周期这个概念，在很多投资者嘴里，通常以”趋势“这个词替代，他们会说”现在是上升／下降“
趋势。但是，真正的趋势常常需要在多个周期（失少两个）之后才能真实展现。如果我们探究
的是真正的趋势，就会发现，上升与下降只不过是一个真理的表象：市场里没有直线，只有波动。
因此一小段时间里的波动表象不是趋势，只有一个以上周期之后所显现的趋势才是真正的趋势。

这也解释了为什么跟涨杀跌的人必然很吃亏，因为他们所看到的并不是实际的趋势，他们看到的和
把握的只不过是幻象而已。如何把自己变成另外一个物种，方法之一很简单，判断趋势的时候，看
至少两个周期。若我们能够看到多个周期，趋势就会明显到不可忽视的地步。

下图是自1800年至2012年的股票和债券的收益比较图（股票指数SPXTRD，债券指数TRUSG 10M）。
起起落落这么多年，股权投资和债权投资的趋势其实是一样的：都在上升；只不过，虽然债权投资
”看起来更稳定“，但涨幅落后于股权投资。

而另外一张图看过去，长期来看，投资黄金、美元什么的，比起投资股票简直是弱爆了。不管你
同不同意，这也是为什么现在越来越多的人认为”股权收益时代来了“的重要原因－其实早就存在了。

进而，几乎一切事物，无论是抽象的、具体的、都有它自己的周期。而它们的周期不大可能一致，
于是，几乎一切的机会和陷阱都隐藏在周期与周期之间的差异上。

[Business Cycle and Stock Market Cycle]
Expansion => Peak => Recession => Button => Recovery => Expansion

[Kubler-Ross model]
Shock => Denial => Frustration => Depression => Experiment => Decision => Integration

于是，我们可以反思这样的现象了：每次巨大技术变革出现的时候，都有一批投资者死在路上。
为什么呢？他们看到了所谓的”趋势“，却忘记了或者不知道真正的趋势需要一个以上的周期才
会正真显现。

刚刚闯入交易市场的人，往往并不知道一个冰冷的事实：在牛市里赔钱的人其实很多，在熊市里赚钱
的人其实也很多。牛市为什么会赔钱？比如有很多人在牛市里担惊受怕追涨杀跌，只因为一个回调就
割肉，又因为不甘心而加上杠杆，到最后落得无法所说的下场。也可以说，坚守自己的理念并镇定自若
应该才是最重要的。

为什么会这样？理由很简单，他们的眼光穿不透周期，于是看不到正真的趋势，能有的只是最肤浅的理解。
于是，一切的行动都相当于是乱打乱撞的”无头苍蝇“，事实上比拿着火把穿过火药厂还可怕，因为他们
拿着火把四处乱窜。

对周期的深入理解，甚至可能影响一个人的性格。在我看来，所谓的不屈不挠，所谓的坚持不懈，更多
的时候，只不过是对自己正在身处某个周期的哪个位置非常了解，所以才更容易做出的决定。为什么
很多的计划最终并不能落实，不了了之？背后最深刻的原因很可能是你当时所身处的自己的生命周期
与世界的种种周期（比如经济周期）都个不相同，于是没有人能帮你具体的定制完全属于你的计划。

人各有各的不同，就好像动物园的动物，它们各有各的生存之道，也各有各的优势和劣势，没有太多实际上
有意义的、通用的、普适优势策略。如果非要挑出来的话，倒是有一个”繁殖能力强是王道“。拿到投资
的领域里来说，就是”赚的方式越多越重要”。太简单了吧？简单到好像没必要教育或学习似的。其实这也是
传递重要知识时所面临的困境“越是重要的东西也是看起来并不相关”。

很多人问我，你现在的主业是什么？我笑嘻嘻的回答，得到专栏作家啊！可视，在这个专栏里，我可以
告诉你为什么我总是用各种各样的方法赚钱－我写专栏，我讲座，我开公司...

因为我必须有投资以外的各种赚钱的方法，才能保证自己确定地能给自己的已投资金额一个“无期徒刑”（至少
是更长的“有期徒刑”）。

如果我只凭投资赚钱，也不是不好，但我觉得自己很可能做不好，因为心态会变，会变得患得患失，会不由
自主地把眼光和注意力从远方挪到眼前，不由自主地变成“近视眼”，看不透周期，看不到趋势，能看到的只剩下
涨跌而已，而是必然会退化为“韭菜”－那又何苦？

对周期的理解，最重要的用处在于它能帮你看到未来，起码看到一小部分。要知道没有人能百分之百清楚地
看到未来，所以那怕一点点，也要比完全看不到强出许多许多，而经年累月地多看到那么一点点，到最后会
形成巨大的认知差异，到最后就会让人们之间产生巨大的差距。

> 1. 理解周期才能有效判断趋势 / 无判断的行为莫过于抛硬币赌博 / 于是坏事可能发生，便一定会发生
> 2. 短期波动的市场表象不是趋势，只有经过一个以上周期形成的趋势才是真正的趋势
> 3. 几乎一切抽象、具体的事物都有自己的周期，于是几乎一切的机会和陷阱都隐藏在不同事物周期与周期的差异上
> 4. 赚的方式越多越重要，避免心态变坏、患得患失、变得短视，从而能确定的给已投资金额判个“无期”
> 5. 对周期的理解，最重要的是能让你看到未来，哪怕一点点的未来，到最后也会形成巨大的认知差异

## 硅谷来信谈投资

**中国仍有投资机会**

之前聊到巴菲特投资原则，如果投资一家公司，就要保证它能够长期稳定增长。我说的
长期，是指20年左右的时间，甚至更长。如果我们已经知道一家公司能做到这一点，那
早几个月，晚几个月买它的股票从长期来看都关系不大，而且买它的股票时，多花20%
的钱，少花20%的钱的影响也不大。

事实上，巴菲特本人买股票常常是买在高位，但是他一持有就是20年甚至更长，因此前面
那点差异就没有什么关系了。相反，如果一支股票，因为购买时多花了20％的溢价，或者
晚买了半年，就变得无利可图，这种股票本身就不值得投资。我想这个道理该不难理解。

不过，在现实生活中，能够把这个道理贯彻下去的人并不多，即使是一些当过跨国公司的
高管，做过投资的大咖，有时也想不清楚这个道理。有位IT大咖，有一年到硅谷忽悠人和
他回国创业，他讲两点：第一，中国机会很多；第二，你们现在要是不和我回去创业，将
来就没有机会了。我不知道你是否听出来他这里面的逻辑有问题。中国有机会的前提是能够
有20年的持续发展，如果只是一年两年的发展，就谈不上是什么机会。好了，假定我们承认
中国有机会，那么就等于承认了它还能发展很长时间，这时早一天晚一天回中国都没有差别。
从美国搬家回到中国不是一件靠冲动可以决定的事情，要谋定而后动。反而是接下来几年
陆陆续续回国的人，有不少做得不错，因为思路理清晰了。

2007年的时候，Z君告诉我，全世界只要投资两个地方就可以了－美国和中国，其它的都不用
考虑。我说，投资美国好理解，毕竟它有200多年高回报的历史可以追溯，投资中国的理由是
什么呢？他讲，《异类》一书中不是说人类历史上有1/5的超级富豪是在美国工业革命中诞生
的吗？中国现在从农业社会向工业社会和商业社会转变，就相当于那个年代。然后Z君又讲，
你自己不是强调《浪潮之巅》的惯性吗，中国的引擎一旦启动，惯性就能推着它往前走。Z君的
看法在过去的十年里完全得到了证实，美国从南北战争后大约花了30年超过英国称为世界第一
经济强国，接着又沿着惯性发展了近20年，这前后半个世纪，虽然美国出现过几次经济危机，
但是整体发展是快速向上的。

中国从1990年开始算起，大约40年左右，可以取代美国成为世界第一大经济体，然后如果政治
稳定的话，还会沿着惯性往前冲一段时间。这中间，也一定会有磕磕碰碰，但是趋势是往上的。
想清了这个道理，就不必担心明天机会是否会消失，因为它的发展是能够持久的。很多人的悲剧
在于，会把短期的困难看成危机，或者把昙花一现看成繁荣。如果大家相信这个时代，要做的事
情其实就简单了，加入这个时代即可。

我在《智能时代》一书中讲，人工智能将是今后20年的趋势，这就如同在过去半个多世纪里，
摩尔定律是趋势一样。既然有20年的时间窗口，那么早一天，晚一天加入进入，其实关系都不大。
今天很多人显得很焦虑，生怕错过这一两年就失去了机会，其实那些看似转瞬即逝的机会不是真的
机会，而只是一次投机。

我们所追求的机会应该是一个长期发展的机会，然后得到长期复合的增长，而不是那种转瞬即逝的
机会，因为后者就算赶上了，也不过是一锤子买卖。其次，一个企业，一个国家，一旦站到了浪潮
之巅，就会顺着惯性往前发展很长一段时间，即使中间有磕磕碰碰，亦无防大趋势。最后，思维的
方式比具体的知识更能决定人的命运。希望你也能成为像Z君这样能够理性思考的人。

> 在中国过去20年里，最后的投资其实不是股市，甚至不是北上广深杭州以外的房市，而是加入到
经济活动中做一件事情。好好做任何一件事情，都有很多回报。

> 1. 长期稳定的增长机会，早一点晚一点进入，多花少花20%的资金，都没实际区别
> 2. 重要的是思维方式，理性思考，思路清晰，不盲从
> 3. 企业国家一旦站到浪潮之巅，就会顺着惯性发展很长一段时间，即使磕碰也不无妨大趋势

**我的投资原则**

1. 不能够盈利或长期不能盈利的企业不投。
2. 挣钱效率低的不投。
3. 参与者不能几方共赢的不投。

当然，我知道大部分读者不投资，但是在做事情时，大家是在用自己的生命在投资。
因此稍微变通，这些投资原则就是做事原则：没有好处的事情不做，效率不高的事情
不做，损人利自的事情不做。

> 效率 盈利 共赢

## 常识和《巴菲特法则》

十几年前，我还会时不时好心向一些朋友转述巴菲特的忠告：不要玩那些危险的股票，
不要在股市上做那些危险的事，不要为了不切实际的回报而冒险，但是几乎没有人听。
我所规劝的周围的朋友不应该算太笨，但是用我朋友段永平的话讲，他们缺乏投资的
智慧。智力、知识和智慧其实是两回事。

我推荐的《巴菲特法则》则是教给人们巴菲特的智慧。作为巴菲特的家庭成员，玛丽
有机会近距离接近这位老人，了解他的生活，看到他在工作之外思考问题的方法论，
这些才是这位股神能够创造奇迹的根本原因。那么巴菲特老爷子智慧的核心是什么呢？
是常识，常识有时恰恰是人们最容易忽略的东西。你可能都听说过，老爷子很少看
《华尔街日报》，从来不相信什么股评（当然也从来不画K线），用段永平的话讲，
不是因为他画K线画不过你们大家，因为他知道这没用。

什么是投资常识？比如股票要“低价买、高价卖”，这就是常识。你可能会说，这不是
废话吗？如果你看看很多所谓的金融学教授、股评家、基金经理都在鼓吹“高价买，
更高价卖出”时（甚至公开嘲笑巴菲特的方法论过时了，因为他们认为金融产品永远
不会再有低价格了，只有更高的价格），当你周围的人都在追捧一钱不值的垃圾股时，
你就知道“低买高卖”是金玉良言，而不是废话了。

巴菲特对经济形式的判断，对一个公司的判断也来自非常朴素的常识。他经常在百货
店逛逛，到街上看看，看看有多少人买东西，看看货架上都放着哪家公司的商品，就
足够得到结论了，这个简单的方法，比经济学家和华尔街根据成千上万数据做出的
估计准得多。

2008年，我告诉我在高盛的财务顾问－苹果公司的市值将很快超越微软，而且会甩
开后者一大截。他问我原因，我说很简单，微软为了赌气，常常在苹果店对面开一家
微软店，你数数里面各自有多少顾客就可以了。我现在的基金合伙人之所以愿意跟
我一起做事情，是因为我在Facebook上市前后准确地告诉他这家公司的价值。后来
他问我是怎么判断的，我说很简单，只要看看优秀的人在公司之间是如何流动的就
可以了。

很多时候，人们最需要的，也是最缺乏的恰恰是基本常识，而不是高深的理论。正是
因为常识看似简单，大家才倾向于忽视它们，这也造就了少数尊重常识的人的成功。
今天相比古代，有知识的人增加了，但是有智慧的，依然很少，希望你能够成为有
智慧的人。

> 越多的规则，越容易混乱，在操作时反而无所适从。账面上涨了就会笃定这个方法，
亏损了又开始后悔是不是用错理论了。其实真正的心法，是越简单越好，越少规则
越好。总结常识，回归简单朴素的原理。

> 美国股市80%的交易早就是机器在做决定了，并没有带来什么额外的好处。主要看
人性，不是技术。真正有经验的人，从来不相信自己能够把握住高点和低点，只有
业余的人才相信。

## 为什么K线分析不管用

股市上的狗屎运常常比我们想象的要多，但是很多人把这种狗屎运叫做水平，把遇到
狗屎运多的人称为股神。如果一个人连续10年每年投资回报率都比大盘好，你是否
觉得他是股神呢？如果你的回答是肯定的，可能数学没有学好。大家随机做股票交
易，每一千个人中就可以出一个这样的“股神”，也就是说在你的朋友的朋友中就有。
现在通信很发达，这样一个人很容易就被朋友的朋友知道了。如果你觉得10年还太
短，有人连续20年每一年做到了这一点，我们是否可以称呼他为“股神”了呢？算一下
概率，这种人在中国，至少有100人，因为每一百万人就会出这样一个。这些人可能
会被请到电视台去做讲座，交大家画K线图。当然，等到你去画，就没有了他的运气。

如果只有一个人知道某一个有用的技术指标，而其他人不知道，那么这唯一的一个人
就能赚钱，如果这个技术指标早已经是所有人都知道的，那么它的作用早已反映在
股市的波动中了，再用它就不灵了。这就是信息论的基本原理。

如果我们说K线等技术指标不灵光，还可以从另一个角度来看这个问题。如果一个人
真正掌握了股市的一个特有的规律，他所做的事情不是开班讲课挣取讲课费，更不是
到电视上去做股市分析，而是闷声发大财。最后一个被发现的“特别灵”的股票技巧
指标（信息）就是著名投资人彼得.林奇发明的PEG（市盈率相对盈利增长比率），
彼得.林奇闷声发大财，从来没有告诉别人，因此创造了股市上不败的神话。但是他
四十岁就收手了，因此很多人认为如果他再炒下去，难免失败。

当然，你可能会问，为什么我们不能像彼得.林奇那样找到一个新的技术指标呢？这种
特别管用的信息在早期股市上是存在的，在今天几乎已经找不到了，因为容易找的早
被人找光了。对于哪些影响力不是很大的指标，可能还存在，但这不是散户可以找到的，
甚至不是绝大多数专业人士可以找到的，因为它们本身太复杂，而且作用也不是很明显，
因此很难找到。如果能找到一大堆，或许组合起来管用，但这是一个非常非常难的
机器学习问题，不要说一般公司做不到，就连Google和微软这样的公司也未必做得到。
事实上，全世界可能只有著名的对冲基金文艺复兴公司敢说它找到了很多有用的信息，
而且能够组合得很好。

和文艺复兴公司每年37%的回报率相比，巴菲特平均每年22%的回报率就小儿科了。不
过这家公司从来不公布它使用的信息，只是闷声发大财。至于其他对冲基金，也是在
大量使用机器学习，或者依赖高频交易挣钱。可以讲，在未来，散户除了买指数基金，
没有更好的交易方式，因为你不可能比得过那些既有资源又有技术的大型对冲基金。

当然，你依然不要指望股民能这么理性，事实上冲动和浮躁正是两百年来股民的心理
特征，这或许就是人性的弱点。但是，如果你了解了K线不管用的原因，放弃了这种无
效的工具，或许投资效益会更好。

> 谁都能赚到钱，只要跟着大盘走就行。别以为自己比市场更厉害就行。

## 巴菲特午餐都聊些什么

巴菲特告诉他，年轻人，我不用告诉你该做什么，因为你们很优秀，自己知道该做什么。
我告诉你一生不该做什么。首先，不要做自己不懂的事情；第二，永远不要做空股票；
第三，永远不要用杠杆投资。

可口可乐既然要花如此巨大的广告费用，为什么没有成为最大的广告公司，只因不做
不懂的事情。公司盲目扩张，什么行业都想插上一脚是不行的。二是不做空股票，做空
股股票的危险在于，股票的上涨从理论上讲是没有尽头的。你一只股票的损失就可能达
到无穷大，以至于全部的资产都用来平仓。巴菲特认为，人在股市上，挣钱赔钱都是
正常的，这不是由人能够决定的，但你的操作方法让你一次损失就满盘皆输，就不能说
智慧了。第三种情况与第二种情况类似，杠杆虽然在股票上涨时放大收益，但下跌时也
会放大亏损。人通常是记吃不记打，好的时候忘乎所以，欲望膨胀；赔了钱则归结于大
盘形势不好，而不去追究自己的过失。在这种情况下使用杠杆，一次就会倾家荡产。

人应该如何平衡进取和稳妥？巴菲特告诉他，人一生不要两次富有。你通过努力，创业
成功，富有起来。接下来去冒险，有成为穷光蛋，但经过坚韧不拔的毅力再次东山再起，
创造辉煌，这就叫两次富有。低谷对于人来说总是有的，但是不应该从富有变成穷光蛋，
这不仅缺乏智慧，也让人的生活变得很糟糕。虽然两次富有的人，即使最终从结果来看
可能钱并不少，但是他的家庭生活，他家人的生活，他的心态，都可能变得不健康，不
愉快。我们看到很多人，在事业上算是成功者，但是在生活上是失败者，原因是他们因为
自己的贪婪，经历了不必要的失败，以至于他们本可以分配给生活和家庭的时间和精力，
都用于东山再起了。因此，巴菲特给他的建议是，少犯错误比多几次成功更重要。

**不做的智慧**

根据段永平先生讲，他从巴菲特身上所学到的不仅仅是价值投资，还有就是什么事情应该
做，什么事情不能做的智慧。在中国过去三十多年里，由于经济快速发展，可以挣钱的机会
非常多，但是有的人成功了，有的人失败了，有的人昙花一现。吴晓波在《大败局》中总结
了中国早期炒股最成功的一批人，除了一两个得到善终，剩下的都没有很好的结果。这些
人共同的问题都在于不懂得什么事情不能做。

对于一个企业，什么事情不能做呢？段永平讲，不熟悉的，不是自己核心业务的，当然不能
做，这时巴菲特的思想精髓。但是很多人不信这个邪，一定要尝试，虽然我们常常鼓励大家
尝试新东西，但是并不意味着在没有搞懂之前就盲目地做。

**投资是艺术，不是技术**

西方科学和技术最显著的特点是可继承性和可叠加性，也就是说总体来看，后面的人学习前
人的知识，会比千人做得更好。但是，巴菲特认为，投资不是一个技术活，而是艺术活，大
部分人学不会，这就如同我们今天没有人敢说钢琴弹得比肖邦或这李斯特好一样。

巴菲特不认可雇一大堆人研究股票的做法，因为在他看来，懂得投资艺术的人非常少，一个
基金找到三五个这样的人，已经算是运气了，如果招聘100人，一大半是懂投资技术而不懂
投资艺术的人，这些人不堪大用。正式因为缺少懂得投资艺术的一流人才，伯克希尔.哈撒韦
只有精力分析少量的股票，这也是为什么它的投资组合中只有很少数的股票。当然，这样它
肯定失去了很多投资机会，不过既然它追求的是少犯错误，而不是失去机会，这种做法当然
合理了。

谈到这些经验时，段永平补充讲到，他拿巴菲特的这种思路把自己公司里的人员梳理一遍，
发现还真是这样。那些无论是懂得产品艺术、营销艺术还是管理艺术的人，都是少数。这也
是他不愿意把自己公司产品线拉得很长的原因，因为没有那么多人才。

> 艺术的关键在与适合自己，适合自己的方法才是最好的方法。因此要冷静思考，细心评判。

**只有价值投资才算是投资**

我们一般把投资方法分为三派（对冲基金不再此列）。第一派是所谓的价值投资，以巴菲特和
美盛集团（Legg Mason）著名投资人比尔.米勒为代表。第二派自称是技术派，不过用巴菲特
和《漫步华尔街》一书作者马尔基尔的话说，是“建立在空中楼阁基础上”的投资，它没有任何
价值依据。对此，巴菲特非常不屑。而马尔基尔虽然不赞同，但是考虑到投资行业人的非理性，
他不否认这种投资人在推动股市上涨和下跌时的作用。中国95%以上的股民和基金经理其实都是
奉行这种投资策略的。这一派投资人鲜有长期成功的，唯一的例外是传奇投资人彼得.林奇，他
找到了一个别人不知道的重要技术指标，然后通过它赚了好几年的钱。

第三派则不相信任何方法，完全听从与市场。它以诺贝尔经济学奖获得者威廉.夏普和著名金融
学家马尔基尔为代表，他们既不赞同建立在空中楼阁上的投资，也不相信什么人能够永远找到别
人看不到的价值。夏普讲，即便一个基金经理或这投资人（比如米勒或巴菲特）连续多年找到了
比大盘表现好的股票，这些基金的价格（或费用）会被迅速提升，以至于它们后来的回报必然接
近与大盘。比较典型的例子就是巴菲特自己的伯克希尔.哈撒韦公司，在过去的10年多里，几乎
完全是根据大盘的走势再走。因此，夏普和马尔基尔都认为，从长远看，买指数基金是最简单而
且高回报的投资，因为这种投资理念摒除了投机的因素，而且有经济增长这个依据。

总的来讲，我比较认同夏普和马尔基尔的观点，如果你知道股市本身是一个带有反馈的系统，从
理论上讲，他们的观点符合系统论。不过，凡事有例外，我们必须承认世界上总有一些有智慧的
人，他们能够看到一些别人找不到的机会，米勒和巴菲特就是这样的人。为什么巴菲特会相信存
在一些别人看不到的机会呢？这就涉及到是否相信个别精英的智慧能够超出大众的智慧，而巴菲
特从骨子里是精英的思维方式，他相信这是可以的。
